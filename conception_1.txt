			- 7 cartes dans la main
			- 1 collection de personnage (Carte)
			- 1 collection de sort (Carte)
		- Nom

		- sockfd
		- Type_Joueur

	Cartes
		Structure de base à faire hériter

			- Référence (alphanumeric, ex: PD-001)
			- Nom DONE
			- Utilisation DONE
			- Coût (int)
			- Stats (Union)
				- HP (uniquement le héros / int)
				- Combat (structure)
					- Attaque (int)
					- Vie (int)
			- Effet*
				- Type d'effet
				- Union

		- Personnage
